## ANACONDAの基本操作

### Anaconda Navigatorを起動
```terminal
conda activate [enviroment_name]
```

### 環境の作成
```terminal
conda create -n [enviroment_name] python=3.7
```

### 環境の削除
```terminal
conda remove -n [enviroment_name] --all
```

### 環境の一覧表示
```terminal
conda env list
```

### 環境の切り替え
```terminal
conda activate [enviroment_name]
```

### 環境の削除
```terminal
conda remove -n [enviroment_name] --all
```

### パッケージのインストール
```terminal
conda install [package_name]
```

### パッケージのアンインストール
```terminal
conda remove [package_name]
```

### パッケージの一覧表示
```terminal
conda list
```

### パッケージのアップデート
```terminal
conda update [package_name]
```

### パッケージの検索
```terminal
conda search [package_name]
```

### パッケージの情報表示
```terminal
conda info [package_name]
```
